JavaScript SPA Pixel Art Website

This project is my personal JavaScript work, a pixel art website. The website uses single-page application (SPA) technology and uses fetch to load template web pages. In the self-developed small framework ZekiCore.js function library, I wrote a function named bindEvent, which is used to bind functions that need to be monitored by addEventListener. In this case, it is used to monitor pushState and replaceState events. When the user clicks on the navigation link, I use pushState and replaceState to update the URL. At the same time, I dynamically load the corresponding template webpage by listening to the trigger events of pushState and replaceState. However, bindEvent is written on the prototype of the function, so in addition to listening to pushState and replaceState events, it can also be bound to all other functions and achieve the listening function.
